# Project Planner Resources

This folder contains templates and logic to help the `project-planner` skill execute its bootstrapping tasks.

## 📄 Project Specification Template
Use this template to provide as much context as possible to the `project-planner`.

```markdown
# Project Name: [Project Name]

## 🎯 Objectives
[Describe the main goal of the project]

## 🛠️ Stack & Infrastructure
- **Frontend:** [React / Angular / Next.js / etc.]
- **Backend:** [Node.js / FastAPI / Go / etc.]
- **Database:** [PostgreSQL / MongoDB / Redis / etc.]
- **Cloud/Infra:** [Docker / AWS / Vercel / GCP / etc.]

## 📐 Architecture
[Describe the preferred architecture pattern]

## 🔗 Integrations, APIs & MCPs
[List any external APIs or required Model Context Protocol (MCP) servers]

## 🛂 Specialized Domain Rules
[List any domain-specific constraints or security standards]

## 🔀 Desired Automations
[List any CI/CD, migration, or developer flows you want to automate]
```
